How to use ExData

Step 1:


Click the 'Begin' button to go to the converter, or if you have any questions on how to use the converter, click on the 'How to use Exdata' accordian to see instructions. If you want a template of the input file, use the download link in the 'Input format (with example file) accordian'



Step 2:


Click 'Go Back' to return to the home page, or you can click on 'Choose files' to select the report documents to be converted.



Step 3:


Once you click on 'Choose files', a file explorer window will open, where you can choose multiple .docx reports which follow the format of the template. Not following the template will result in the converter failing.



Step 4:


After you have selected the report documents to be converted, the files selected will appear in the queue. Don't see the file you want? Click on 'Try again' to clear the queue, and repeat from Step 2.



Step 5:


Click on 'Convert' to strt the conversion of multiple report .docx files to a single Excel .xlsx file. If you selected multiple files, each .docx file will be converted to a seperate sheet in the Excel file. The sheet name would be the name of the report .docx file. The Excel output would begin to download as soon as the conversion is complete. Keep in mind that larger files take longer to convert.
